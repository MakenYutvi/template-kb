# Query Workflow

Используй этот workflow, когда пользователь задает вопрос по базе знаний.

## Шаги

1. Прочитай `README.md`, `wiki/index.md` и `wiki/current-status.md`.
2. Найди релевантные source-файлы в `raw/` или summary в `wiki/`.
3. Отдели факты, выводы и гипотезы.
4. Если информации недостаточно, скажи это прямо.
5. Дай короткий ответ и укажи, какие источники использованы.

## Definition of Done

- Ответ опирается на найденные источники.
- Неуверенность явно обозначена.
- Не пересказаны секреты или чувствительные детали без необходимости.
