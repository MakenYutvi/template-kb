# Рабочие встречи

Optional source-область для рабочих meeting packages: транскрипты, заметки к
записям, summaries и writeback candidates.

Перед переносом durable context в `wiki/people/` или другие wiki-страницы
используйте `wiki/workflows/meeting-ingest.md`.

Соблюдайте правила компании, NDA и privacy constraints. Не храните secrets в
plaintext.
