# Личные встречи

Optional source-область для личных meeting packages: транскрипты, заметки к
записям, summaries и writeback candidates.

Перед переносом durable context в `wiki/people/` или другие wiki-страницы
используйте `wiki/workflows/meeting-ingest.md`.

Чувствительные детали оставляйте в `raw/`, если не нужен короткий безопасный
summary.
